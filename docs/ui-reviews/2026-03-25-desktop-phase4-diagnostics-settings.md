- User flow summary: primary surface には top bar badge と topic ごとの `connectionLabel / peerCount / lastReceivedLabel` だけを残し、`expected / missing / raw error / status detail` は settings drawer 側へ集約した。settings drawer は `Connectivity`, `Discovery`, `Community Node` の 3 panel に固定し、shared primitive として metric grid、diagnostic list、action row、inline message 付き editor field を導入した。`Discovery` は `env_locked` read-only と `Save Seeds / Reset` の挙動を維持し、`Community Node` は base URL editor と node card を維持したまま `Authenticate / Consents / Accept / Refresh / Clear Token` を card 単位で整理した。
