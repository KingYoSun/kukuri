- User flow summary: primary nav から `Channels`, `Live`, `Game`, `Profile` に直接移動でき、settings drawer は `Connectivity`, `Discovery`, `Community Node` に限定した。private channel panel は channel list と selected detail を同じ surface に集約し、選択した channel を `timeline scope` と `compose target` の正本にする。live と game は `create form + list` に整理し、owner gating と pending disable を row 単位で出す。profile editor は main workspace に移し、保存は field state の更新で表現する。
