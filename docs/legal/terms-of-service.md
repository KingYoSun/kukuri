# kukuri 利用規約（ドラフト）

Last updated: 2026-06-25
Legal bundle version: 1

> これはドラフトであり、法的助言ではありません。最終的な運用上・法的判断は、適切な専門家・規制当局への相談のもとで行ってください。
> 本文書は、kukuri デスクトップアプリ（client）自体に対する app-level の利用規約です。各 community node が個別に提示する利用規約（per-node consent）とは別建てです。

## 第1条（kukuri の位置づけ）

kukuri は P2P を基盤とする分散型アプリケーションです。ネットワーク全体を統治する中央権限は構造的に存在しません。ユーザーの identity / profile / social graph は特定の community node に所有されず、特定ノードの停止・変更によっても失われません。

## 第2条（禁止事項）

ユーザーは、以下の目的または態様で kukuri を利用してはなりません。

- 法令に違反する目的での利用。
- 児童の性的搾取（CSAM / CSE）その他の重大な権利侵害コンテンツの作成・共有・流通。
- 他者の権利（知的財産権・プライバシー・名誉等）を侵害する行為。
- ネットワーク、community node、relay、その他補助機能の妨害。
- マルウェア・フィッシングその他の不正・有害な目的での利用。

## 第3条（開発者・ノード運営者の免責）

- kukuri の開発者は、P2P 基盤上で動作するソフトウェアを提供するものであり、ネットワーク全体・他ユーザー・各 community node の活動について責任を負いません。
- community node 運営者は、その運営者が実際に関与した補助機能の範囲でのみ責任を負い、kukuri network 全体・他ノードの活動については責任を負いません。
- kukuri は現状有姿（as-is）で提供され、明示または黙示を問わず、特定目的への適合性その他のいかなる保証も行いません。
- 適用法令が許容する最大限の範囲で、開発者およびノード運営者は、本ソフトウェアの利用または利用不能から生じるいかなる損害についても責任を負いません。

## 第4条（ユーザーの責任）

ユーザーは、自らが作成・共有・取得するコンテンツおよび行為について、自身で責任を負います。

## 第5条（本規約の変更）

本規約は更新されることがあります。重要な更新があった場合、アプリは更新後の規約を再表示し、再同意を求めます。同意しない場合、ネットワーク接続を伴う機能は利用できません。
